== はじめに

このアーカイブには「Ruby de XML」で使用しているスクリプト、
第2章の章末に掲載されてあるXPathの関数リファレンスのHTML版、
付録のREXML 2.4.2のツリーAPIのリファレンスのHTML版が収録されています。

収録されているスクリプト、リファレンスはRubyと同じ
ライセンスで使用、再配布可能です。

収録されているスクリプト、リファレンスの利用は
各自の責任で行って下さい。
もし、なんらかの損害、障害等が発生した場合でも私は
一切の責任を負わないものとします。

== 注意

スクリプト、リファレンスともに本書で掲載されているものと
異なる部分があります。

スクリプトでは、パスが異なっていたり、本書中では紙面の都合での
折り返しを回避するために使用している変数を使用していなかったりします。

リファレンスでは、画像がテキストベースでの図になっていたり、
rd2でリンクを作成するために"REXML::Element"が"REXML::Elementクラス"
等のように"クラス"や"モジュール"が付いていたりと一部名前が
変更されている箇所があったり、本書中では紙面の都合での折り返しを
回避するために使用している変数を使用していなかったりします。

どちらの場合も本書で掲載しているものと本質的には変わりませんが、
本書で掲載しているものとまったく同じではないことをご了承下さい。

== アーカイブ

アーカイブはWindows用とUNIX用が用意されています。

違いは以下の通りです。

   +----------+-------------+-----------+
   |          |  文字コード | 改行コード|
   +----------+-------------+-----------+
   |Windows用 |  SJIS       | CR+LF     |
   +----------+-------------+-----------+
   |UNIX用    |  EUC-JP     | LF        |
   +----------+-------------+-----------+

== リファレンス

収録されている二つのリファレンスは同じディレクトリ内にある
base.cssというファイル内に記述されてあるCSSを読みこんでいます。
base.cssを編集することにより表示の仕方を変えることができます。

関数/メソッド一覧はDOMを用いてJavaScriptで作成されています。
関数/メソッド一覧を生成したくない場合は、HTMLファイルのbody要素の
最後の子要素のscript要素内の"make_function_list();"または
"make_method_list();"とある行の行頭に"//"をつけてコメントアウトするか、
ブラウザのJavaScriptの機能をOFFにして下さい。
